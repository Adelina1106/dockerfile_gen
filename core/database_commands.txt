Pentru schimbare parola root(nu mai merge):
sudo mysql -u root
ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY 'new_password';
FLUSH PRIVILEGES;
exit;
Conectare iar: mysql -u root -p
